# INSTRUCTION!!!!!!!!!!!!!!

Description
Een muziek bot die in een discord call muziek kan draaien.

## dependencies
> npm i discord.js
> npm i -g nodemon
> npm i spotify-api.js@latest

## starten
> nodemon
>
> ##functionalitijten
###muziek afspelen
	- nummers aanzetten
	- pauzeren/afspelen
	- herhalen

###statistieken
	- top 5 meest afgespeelde nummers van bot
	- top 10 wereldwijd

###nummer info
	- tekst weergeven
	- general info weergeven (naam, artiest, streams, etc)

  ##opslag
de bot houdt een list bij van zijn geschidenis. de URLs van de laatste 5 (of meer als gewilt is) nummers worden opgeslagen.

##Media
Muziek die de bot gaat draaien
